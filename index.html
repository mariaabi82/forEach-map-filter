Métodos de array

FOR EACH

El bootstrap del for

Ejecuta la función a cada elemento

Recibe un parámetro obligatorio (cuyo nombre elegimos) y puede recibir un 2do parámetro, que es el índice

MAP

Recorre 1 array y por cada elemento aplica una función, igual que el forEach

Recibe una función como parámetro y puedo tener otro parámetro opcional, el índice

La única diferencia con forEach es que map me devuelve un array

La función que está dentro de map debe devolver algo por sí misma para que map me pueda devolver un nuevo array

Después de la flecha está lo que JS asume que es lo que tiene que retornar

FILTER

Retorna un array con los elementos que cumplan con la condición que nosotras establezcamos

Lo que retorne el filtro tiene que poder retornar un booleano (claramente, porque estamos estableciendo una condición
